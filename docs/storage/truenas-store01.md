# TrueNAS Mini-R (store01)

store01 is a TrueNAS Mini-R at the fairy site dedicated to media storage,
Time Machine backups, and SMB shares.

## Hardware

- **Form Factor**: 2U rackmount (U3–4 in r02)
- **Drives**: 12x 26TB Seagate Exos
- **Pool Layout**: 2x 6-wide raidz2 vdevs
- **Usable Capacity**: ~208TB
- **Uplink**: 10G copper to r02-tor01 (Netgear XS724EMv2)

## Uses

- Media libraries (Plex, Jellyfin)
- Time Machine backups
- SMB shares
